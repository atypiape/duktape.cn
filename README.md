# Duktape 中文网

Duktape 中文网是 [duktape.org][duktape.org] 的非官方中文版本，与官网源自同一套代码，既来自 GitHub 上的开源项目 [Duktape][github-duktape] 下的 [website][github-duktape-website] 目录。
